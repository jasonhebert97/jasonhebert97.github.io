User Interfaces & User Experience Projects

The following projects have been presented in an order that follows the typical timeline for developing a user-centric interface. Personas & Storyboards highlights the first step with developing a specific user to refer to during development. Interface Redesign follows showing the process for developing both lo- and hi-fidelity mockups. Find a Cool Cat is the third step of creating an interactive and deployable interface. And lastly, A/B Testing investigates methodologies for evaluating how successful an interface is using data.

Personas and Storyboards

A project showcasing familiarity with curating personas and storyboard users for a given interface.
Interface Redesign

A project showcasing familiarity with lo-fidelity and hi-fidelity mockups.
Find a Cool Cat

A project showcasing familiarity and ability to create an interactive webpage using HTML, CSS, and React.
A/B Testing

A project showcasing familiarity with A/B testing technologies and methodologies.
