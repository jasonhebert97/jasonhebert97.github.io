Introduction

An autonomous flying drone constructed and coded for the class 'Introduction to Robotics' (CSCI1951R) at Brown University. The goal was to assemble a drone from scratch and write the appropriate software for enabling autonomous flight. Explored topics include safety, hardware, sensors, actuators, Unscented Kalman Filters, PID control, particle filters, and simultaneous localization and mapping (SLAM).Programmed using Python and Robot Operating System (ROS).
